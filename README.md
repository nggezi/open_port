# Open Port

OpenWrt 端口转发一键工具

## 功能

- 自动获取路由器 LAN IP（支持 `/24` 等掩码格式）
- 一键开启/关闭多个端口的 WAN→LAN 端口转发
- 支持 TCP/UDP 协议
- 操作失败自动回滚

## 使用前提

- OpenWrt 系统
- 已安装 `uci` 和 `firewall`（默认已安装）
- 通过 SSH 连接到路由器

## 使用方法

1. 将脚本上传到 OpenWrt 设备（如 `/root/openport.sh`）
2. 添加执行权限：`chmod +x /root/openport.sh`
3. 运行脚本：`sh /root/openport.sh`

## 配置

默认转发端口：`7681`、`7766`、`7676`

如需修改，编辑脚本第 4 行：

```sh
PORTS="7681 7766 7676"
```

## 菜单说明

| 选项 | 功能 |
|------|------|
| 1 | 开启转发（从 WAN 到 LAN 的 DNAT） |
| 2 | 关闭转发（删除规则） |
| q | 退出 |

## 安全提示

- 开启转发后，端口将暴露到外网，请确认来源可信
- 建议仅在需要时开启，使用后及时关闭
